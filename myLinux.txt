#Linux

가상머신 // 하드웨어를 소프트웨어적으로 구현해서 그 위에서 운영체제가 작동하도록 하는 기술
ex. orcle virtualBox

가상머신을 사용하는 이유
- 다른 운영체제를 사용해야 하는 경우 (맥OS에서 윈도우, 윈도우에서 리눅스)
- 독립된 작업공간이 필요한 경우(바이러스 회피, 백업(스냅샷))
- 하나의 머신에서 여러명에게 운영체제 환경을 제공
*웹 포스팅 : 하나의 웹 서버 안에 하나의 운영체제가 깔려있고 그 운영체제에 있는 여러 가지 계정들을 만들어서 사용자들에게 계정을 주는 것
 가상 포스팅 : 하나의 컴퓨터 안에 여러 개의 운영체제를 설치해서 각각의 사용자에게 운영체제를 통째로 제공해주는 것
